Steganography Algorithms

Multiple advanced techniques for data hiding and detection

LSB (Least Significant Bit)

Hides data in the least significant bits of pixel values. Simple, fast, and widely used for basic steganography with minimal visual impact.

DCT (Discrete Cosine Transform)

Embeds data in frequency domain coefficients. More robust against compression and provides better security than spatial domain methods.

DWT (Discrete Wavelet Transform)

Uses wavelet decomposition for data hiding. Offers excellent imperceptibility and resistance to various image processing attacks.
